# ResumeForge AI — Elite ATS Resume Builder

ResumeForge AI is a premium, recruiter-grade, ATS-optimized resume builder designed to achieve 95+ compatibility scores against modern applicant tracking systems (such as Workday, Greenhouse, Lever, Taleo, Ashby, and BambooHR).

## 🚀 Features

- **95+ ATS Score Target**: Structural format strictly complies with parsing rules (no custom icons, tables, columns, graphics, or complex templates).
- **Recruiter Friendly & Clean Layout**: Focuses on clear hierarchies using industry-standard headings.
- **Dynamic Optimization & Score Calculator**: Real-time diagnostic panel highlighting missing fields, impact statements, and keyword alignment.
- **Interactive Experience & Project Customizer**: Metric-driven bullet assistance (Action + Tech + Scale + Impact formula).
- **Instant Local Export**: Pure client-side PDF export utilizing `html2pdf.js` with exact standard printing dimensions.
- **Vibrant Dark-Themed UI**: Premium dashboard aesthetics with glassmorphic elements and particle background.

## 🛠️ Tech Stack

- **Core**: HTML5, Vanilla CSS3 (Custom Properties / Design Tokens)
- **Logic**: Vanilla JavaScript (ES6)
- **PDF Compiler**: html2pdf.js

## 💻 How to Run Locally

You can launch the static application using Python's built-in server:

```bash
# Using Python 3
python -m http.server 8080
```

Once running, open: [http://127.0.0.1:8080](http://127.0.0.1:8080)
